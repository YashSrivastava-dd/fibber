# FIberiseFit Wonder Theme - Next.js Clone

A Next.js recreation of the Shopify Wonder theme (Root preset) for the FIberiseFit industry.

## Features

- **Sticky Header** with logo, navigation, search, and cart
- **Homepage Sections**:
  - Full-width Hero with CTA
  - Brand value/FIberiseFit messaging
  - Featured products/collections grid
  - Image + text storytelling
  - Benefits/ingredients/science section
  - Testimonials with video support
  - Secondary featured products
  - Editorial/lifestyle imagery
  - Email signup/CTA block
- **Collection Pages** with filters and sorting
- **Product Pages** with media gallery, variants, and accordion tabs
- **Cart Drawer** with slide-out functionality
- **Multi-column Footer**

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
- Zustand (state management)
